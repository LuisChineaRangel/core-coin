Detailed Description

Represents a solution for money change.

It stores the amount of coins for solution and a pair of vectors containing all types of coins needed and their respectives amounts.

Example

Change solution;
solution.insertCoin(ONE_CENT);
std::cout << solution;
ONE_CENT
const Coin ONE_CENT(0.01)
All types of coins existing in the European Union.
